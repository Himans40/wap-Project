About Us

Who We Are

BookFinder is a simple web application built to help readers discover books from Google's massive book database. Whether you're looking for programming guides, novels, or academic textbooks — we've got you covered.

Our Mission

Our mission is to make book discovery easy and accessible for everyone. Just type a keyword, filter by author or year, and explore thousands of books instantly.

Technology Used

This project is built using HTML, CSS, and vanilla JavaScript. It fetches real-time data from the Google Books API using async/await and the Fetch API.

The Team

This project was developed as a first-year B.Tech project to learn and practice web development fundamentals including DOM manipulation, API integration, and responsive design.

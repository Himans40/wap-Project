# 📚 Open Library Book Finder

## 📌 Project Overview

This is a web application that allows users to search for books and explore a vast digital library. Users can view book details, cover images, and organize results efficiently.

## 🎯 Purpose

The goal of this project is to help users easily discover books and explore information about them while practicing modern React development concepts.

## 🌐 API Used

Open Library API - https://openlibrary.org/developers/api

## 🚀 Features

* Search books by title or author
* Display book results with cover images
* Sort books by publish year
* Handle empty search results gracefully
* Debounced search for better performance
* "I've Read This" button with reading counter

## 🛠 Technologies Used

* JavaScript (ES6)
* CSS 
* Fetch API

## ▶️ Setup Instructions

* Clone the repository
* Run `npm install`
* Run `npm start`
